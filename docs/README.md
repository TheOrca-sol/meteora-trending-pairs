## üé® Design Recommendations

### For Print Materials (ONE_PAGER.md):

**Layout:**
- Use 2-column layout for better readability
- Keep font size 10pt minimum
- Use brand colors consistently
- Include logo at top

**Elements to Add:**
- QR code (bottom right)
- Company logo (top)
- Social media icons
- Professional photos/screenshots

**Print Specs:**
- Size: 8.5" x 11" (Letter) or A4
- Format: PDF for printing
- Resolution: 300 DPI minimum
- Color mode: CMYK for print

### For Presentation (PITCH_DECK.md):

**Slide Design:**
- Use consistent template
- Limit text per slide (max 6 lines)
- Large, readable fonts (24pt+)
- High-quality images only

**Color Scheme:**
- Primary brand color
- Dark background for readability
- High contrast text
- Consistent throughout

**Tools to Convert Markdown to Slides:**
- Marp (recommended)
- reveal.js
- Google Slides (manual import)
- Keynote/PowerPoint (manual)

## üí° Best Practices

### Writing Style:
- Keep it conversational
- Use active voice
- Short sentences and paragraphs
- Bullet points over long text
- Emphasize benefits, not just features

### Technical Content:
- Explain jargon
- Use analogies for complex concepts
- Include examples
- Add visuals where possible
- Link to deeper resources

### Calls to Action:
- Be specific ("Try it now" not "Learn more")
- Create urgency
- Make it easy (one click)
- Multiple CTAs per document
- Track which CTAs convert
