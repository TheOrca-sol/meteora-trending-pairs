## Step 2: Browse Pools

### Understanding the Dashboard

When you first load the page, you'll see a table with columns:

| Column | What It Means |
|--------|---------------|
| **Pair Name** | The two tokens in the pool (e.g., SOL/USDC) |
| **30m Fee Rate** | Annualized fee rate based on last 30 minutes |
| **24h Fees** | Total fees earned in last 24 hours (in USD) |
| **APR** | Annual Percentage Rate (yearly return %) |
| **TVL** | Total Value Locked (total liquidity in pool) |
| **Volume 24h** | Trading volume in last 24 hours |
| **Buy/Sell** | Distribution of buy vs sell transactions |

### Basic Navigation:

**Scroll:** Use your mouse or trackpad to scroll through pools

**Sort:** Click any column header to sort by that metric
- First click: Sort ascending
- Second click: Sort descending
- Example: Click "APR" to see highest APR pools first

**Refresh:** Click the refresh button at the top to manually update data

**Auto-Refresh:** Toggle "Auto-Refresh" to update every 60 seconds automatically

## Step 3: Use Filters

### Finding Pools That Match Your Criteria

Located at the top of the dashboard, filters help you narrow down 4,000+ pools to what matters to you.

### Common Filter Examples:

**Example 1: Find High-APR Opportunities**
1. Find "Minimum APR" field
2. Enter: `50`
3. Click "Apply" or press Enter
4. Now you only see pools with APR > 50%

**Example 2: Find Large, Liquid Pools**
1. Set "Minimum TVL": `50000` (for $50k+)
2. Set "Minimum Volume 24h": `10000` (for $10k+)
3. Apply filters
4. Safer, more established pools appear

**Example 3: Find Trending Small Caps**
1. Set "Maximum TVL": `10000` (under $10k)
2. Set "Minimum 30m Fee Rate": `5`
3. Apply filters
4. Discover early-stage opportunities

### Available Filters:

- **Search** - Find pools by name or token address
- **Minimum 30m Fees** - Filter by 30-minute fee earnings
- **Minimum Volume 24h** - Filter by trading volume
- **Minimum TVL** - Filter by total liquidity
- **Minimum APR** - Filter by annual returns
- **Hide Blacklisted** - Remove flagged pools (recommended)

### Tips:
- Start with broad filters, then narrow down
- Combine multiple filters for precise results
- Save your favorite filter combinations for later
- Clear filters to see all pools again
